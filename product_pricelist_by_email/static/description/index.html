Product Price List Report

The primary purpose of this app is to admin user can receive all product price list on every Monday in an email with attachments.

  • Admin users can see product price list on every monday.

  • Users will need product price list to access the email template: Enable developer mode Go to Settings > Technical > Email > Templates and find "All Pricelist - Send by Email"

Prerequisites before Installation

No prerequisites required before installation.

Configurations

Multiple Pricelist Report

  • User can print multiple product price report manually from List View.

Single Pricelist Report

  • User can print product price report manually from Form View.

PDF Report of Product Pricelist.

  • Generated PDF Report of Products Pricelist which active for current month.

Scheduler to send an Email to All Sales Manager

  • Here, Using this Scheduler all sales manager can got one Email with Pricelist report PDF.
